Step-by-Step Guide to Coding Tasks

  1. Task 1: Fetch Data from an API

    In this task, we will use a C library called libcurl to fetch data from an API. An API is like a waiter that brings data from a restaurant (the server) to you (the client).

    1. First, install the libcurl library on your computer.
    2. Next, write a simple C program to make a GET request to a public API that provides placeholder posts.

    Code Explanation:

    #include <stdio.h> // This includes the standard input/output library
#include <stdlib.h> // This includes the standard library for general functions
#include <curl/curl.h> // This includes the libcurl library for handling URLs

// This function is called whenever data is received from the API
size_t write_callback(void *ptr, size_t size, size_t nmemb, void *userdata) {
    printf("%s", (char *)ptr); // Print the received data to the console
    return size * nmemb; // Return the size of the data
}

int main() {
    CURL *curl = curl_easy_init(); // Initialize a curl session
    if (curl) {
        curl_easy_setopt(curl, CURLOPT_URL, "https://jsonplaceholder.typicode.com/posts"); // Set the URL to fetch data from
        curl_easy_setopt(curl, CURLOPT_WRITEFUNCTION, write_callback); // Set the function to handle the received data
        curl_easy_perform(curl); // Perform the request to fetch the data
        curl_easy_cleanup(curl); // Clean up the curl session
    }
    return 0; // Return 0 to indicate successful completion
}
